# 文本编辑器运行说明

## 方法一：使用 Maven Exec 插件运行（推荐）

这是最简单的方法，我已经在 `pom.xml` 中配置了 exec 插件。

### 步骤：

1. **打开终端/命令行，进入项目目录：**
   ```bash
   cd demo
   ```

2. **运行程序：**
   ```bash
   mvn exec:java
   ```

   或者：
   ```bash
   mvn exec:java -Dexec.mainClass="com.editor.App"
   ```

## 方法二：先编译再运行

### 步骤：

1. **编译项目：**
   ```bash
   cd demo
   mvn clean compile
   ```

2. **运行程序：**
   ```bash
   java -cp target/classes com.editor.App
   ```

## 方法三：打包成 JAR 后运行

### 步骤：

1. **打包项目：**
   ```bash
   cd demo
   mvn clean package
   ```

2. **运行 JAR 文件：**
   ```bash
   java -cp target/demo-1.0-SNAPSHOT.jar com.editor.App
   ```

## 使用示例

程序启动后，你会看到：

```
文本编辑器 v1.0
输入 'exit' 退出程序

> 
```

然后可以输入命令，例如：

```
> init test.txt with-log
新缓冲区已创建: test.txt

> append "Hello World"
文本已追加

> show
1: Hello World

> save
文件已保存: test.txt

> exit
```

## 常用命令

### 工作区命令
- `load <文件路径>` - 加载文件
- `save` - 保存当前文件
- `save all` - 保存所有文件
- `init <文件路径> [with-log]` - 创建新文件
- `close` - 关闭当前文件
- `edit <文件路径>` - 切换到指定文件
- `editor-list` - 显示所有打开的文件
- `undo` - 撤销
- `redo` - 重做
- `exit` - 退出程序

### 文本编辑命令
- `append "文本"` - 追加文本
- `insert <行号:列号> "文本"` - 插入文本
- `delete <行号:列号> <长度>` - 删除字符
- `replace <行号:列号> <长度> "文本"` - 替换文本
- `show` - 显示全部内容
- `show <起始行:结束行>` - 显示指定范围

### 日志命令
- `log-on` - 为当前文件启用日志
- `log-off` - 关闭当前文件的日志
- `log-show` - 显示当前文件的日志

## 注意事项

1. 确保已安装 Java 21 或更高版本
2. 确保已安装 Maven
3. 文件路径可以使用相对路径或绝对路径
4. 带空格的文本参数需要用双引号包裹
5. 程序退出时会自动保存工作区状态到 `.editor_workspace` 文件
